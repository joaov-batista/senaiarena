# 🚀 SENAI Star Battle

![Status do Projeto](https://img.shields.io/badge/status-em_desenvolvimento-green)
![Linguagem](https://img.shields.io/badge/language-HTML/JS-orange)

Uma experiência arcade nostálgica que funde as mecânicas clássicas de **Pong** com o dinamismo dos jogos de **Space Invaders**. O diferencial? Os heróis e vilões desta batalha espacial são os próprios alunos do SENAI!

Este projeto foi desenvolvido com o objetivo de praticar lógica de programação, manipulação de DOM e sistemas de colisão em JavaScript puro.

## 🕹️ O Jogo
No **SENAI Star Battle**, você controla uma nave personalizada com o rosto dos desenvolvedores. O objetivo é defender seu território enquanto ataca os oponentes em uma mecânica híbrida que exige reflexos rápidos e estratégia.

### Principais Funcionalidades:
- **Movimentação Fluida:** Sistema de controle otimizado via teclado.
- **Customização Visual:** Sprites baseados em fotos reais para uma experiência personalizada.
- **Sistema de Pontuação:** Acompanhamento de performance em tempo real.
- **Mecânica Híbrida:** Mistura elementos de rebatida (Pong) com tiro (Shooter).

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estruturação do canvas e interface.
- **CSS3:** Estilização, animações de fundo e layout responsivo.
- **JavaScript (ES6+):** Engine do jogo, lógica de colisão e gerenciamento de estados.

## 🚀 Como Rodar Localmente

Para testar o jogo em sua máquina, acesse o site https://senaiarena.vercel.app/ ou siga os passos abaixo ou acesse:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/joaov-batista/senai-star-battle.git
   ```

2. **Acesse a pasta do projeto:**
   ```bash
   cd senaiarena
   ```

3. **Abra o jogo:**
   Como o projeto utiliza tecnologias web nativas, basta abrir o arquivo `index.html` em qualquer navegador moderno ou utilizar a extensão **Live Server** no VS Code para uma melhor experiência.
